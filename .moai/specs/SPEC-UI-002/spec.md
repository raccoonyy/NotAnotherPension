# @SPEC:UI-002: 가로 스크롤 결과 테이블

## HISTORY

### v0.0.1 (2025-12-06)
- **INITIAL**: 결과 테이블 UI 명세 작성
- **AUTHOR**: @Alfred
- **REASON**: 연도별 예상 수령액 비교 테이블 구현

---

## 개요

계산 결과를 연도별로 가로 스크롤 테이블로 표시합니다.
국민연금과 개인연금 상품별 예상 수령액을 세후 기준으로 비교할 수 있습니다.

---

## Ubiquitous Requirements (기본 요구사항)

- 시스템은 연도별 예상 소득을 가로 스크롤 테이블로 표시해야 한다
- 시스템은 국민연금 예상 수령액 (세전/세후)을 표시해야 한다
- 시스템은 개인연금 상품별 예상 수령액 (세전/세후)을 표시해야 한다
- 시스템은 금액을 한국 원화 형식으로 포맷해야 한다

---

## Event-driven Requirements (이벤트 기반)

### UI-002-E01: 테이블 렌더링
- **WHEN** renderResultTable(result)가 호출되면, 시스템은 결과 테이블을 생성해야 한다
- **WHEN** 테이블이 화면 너비를 초과하면, 시스템은 가로 스크롤을 활성화해야 한다

### UI-002-E02: 세금 정보 표시
- **WHEN** 사용자가 세금 항목에 마우스를 올리면, 시스템은 계산 근거를 툴팁으로 표시해야 한다

---

## Constraints (제약사항)

- 모든 금액은 만원 단위로 표시한다
- 세전/세후 금액을 구분하여 표시한다
- 모바일에서도 가로 스크롤이 가능해야 한다
